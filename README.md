# Project Description

A machine learning project that implements a **k-Nearest Neighbors (k-NN)** classifier to predict health outcomes using preprocessed healthcare data.

##  Project Overview

This project explores the use of a KNN classifier on a real-world healthcare dataset. It includes:

- Data preprocessing and cleaning
- Feature scaling and pipeline construction
- Model training using k-NN
- Evaluation using cross-validation
- Metric reporting and results saving
